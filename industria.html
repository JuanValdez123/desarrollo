La transformación digital ha adquirido un papel muy importante, ahora es la responsable de cambiar la industria y adaptarla a una industria conectada. Es un proceso que requiere de una gran concienciación personal y unificada de todos los componentes de una compañía.

Ahora las compañías de fabricación industrial están utilizando una tecnología avanzada que pasa de una producción en masa a una producción personalizada, y está sucediendo a un ritmo acelerado.

Internet de las cosas

El Internet de las cosas se fundamenta en la interacción entre los equipos de adquisición de datos con la nube. El Internet de las cosas es la revolución entre objetos y personas, juntándolos de una forma en la que se puede interactuar entre sí. La ventaja que ofrece con esta conexión es obtener datos en tiempo real.

Node-RED

Node red es una herramienta de programación desarrollada por IBM que está construida en Node.js , que permite conectar varios dispositivos por la simplicidad que trae y es por eso que node red está diseñado o se le está dando su mejor uso para dar soluciones iot de una manera fácil y sencilla, ya que la programación se hace a través de flujos, no es necesario dominar un lenguaje de programación de alto nivel, es un software open source lo cual nos indica que no hace falta pagar alguna licencia para utilizarla, y tiene una interfaz amigable para hacer las diferentes tareas que queremos realizar.

Cloud Computing

Cloud Computing actualmente es una tecnología que ofrece servicios a través del internet, basado en servidores accesibles desde cualquier lugar y en tiempo real. Tiene 10 como característica principal el acceso, almacenamiento y gestión de los datos obtenidos por los dispositivos.

Plataforma Web

Realizamos la plataforma web conteniendo únicamente los indicadores que sean mas importante de un proceso.

La plataforma es visualizado con el fin que la información que se requiera, se la pueda presentar en una forma entendible al usuario, en donde los datos se van actualizando constantemente.

Gracias a la plataforma web, se puede acceder al proceso monitoreado desde cualquier dispositivo conectado a internet.
